# Example Configurations

Each of these example configurations comes complete with Companion Configuration file (ending with .companionconfig), an XML configuration for Scoreboard Engine and a basic HTML overlay for use in OBS or other web source.   The HTML overlays for example configurations are included in the engine distribution in the `pages` folder.

Refer to the main [README](/README.md) Quickstart guide for getting up and running quickly with an example configuration.
